# Idle Clicker / Idle Game with Angular

This is an idle game built with Angular, designed to showcase game mechanics, world progression, and UI interactions in a web app environment. Unlike a traditional clicker, this game runs in the background, giving players resources and progress even while they’re idle.

**Live Demo:** [Play the game](https://idle-loader.netlify.app/)

## Features

- Multiple worlds (Classic, Arcade, etc.) with unique player states.
- Idle resource generation with timers and animated loaders.
- Offline rewards: progress continues while away.
- Buyable spinners with cost calculation and max-purchase logic.
- Settings to toggle animations for a smoother or faster gameplay experience.
- Dynamic loading screens for immersive game feedback.

## Tech Stack

- Angular (Standalone Components only, no NgModules)
- RxJS for timers and reactive state management
- LocalStorage for persistent save/load system
- Netlify for instant deployment

## Purpose

- Learn and experiment with Angular state management and component architecture.
- Explore idle game design concepts like timers, auto-progress, and world-specific states.
- Demonstrate modern frontend practices in a fun, interactive project.
